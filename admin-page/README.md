## 2024 1학기 캡스톤 프로젝트 (메디앙 시스템 - 어드민페이지)

### 주제
- 다중 PC 환경 모니터링 및 원격 채팅 기술

### 사용 도구
- Code : React
- UI : Mantine, React-Icon
- 로그인 : Firebase

### 실행 방법
- 'admin-page' 폴더로 이동 후 아래 아래 명령어를 입력
```
npm install
npm start
```

### 배포 후 수정 파일
```
.env 파일

REACT_APP_API_BASE_URL = http://34.64.47.84
```
- 사용할 기본 URL로 변경 필요
